# piRTLdrivers

Collection of recent drivers for use with wifi adapters that require 8822 Realtek Drivers

Used to install on RaspPi source http://downloads.fars-robotics.net/

Placed these files here incase required and http://downloads.fars-robotics.net/ down.

Instructions

1. Download
2. xzvf tar.gz
3. run install.sh
4. Done
